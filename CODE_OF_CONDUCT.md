# Code of Conduct

This project is intended to be a welcoming space for Indonesian creators, students, solo founders, small business owners, and beginner builders.

## Expected behavior

- Be respectful and patient
- Help beginners without judgment
- Give constructive feedback
- Use inclusive language
- Focus on learning and practical improvement

## Unacceptable behavior

- Harassment or personal attacks
- Gatekeeping beginner contributors
- Spam or promotional abuse
- Misleading claims
- Disrespectful comments about skill level, background, language, or location

## Enforcement

Project maintainers may remove comments, issues, pull requests, or contributions that violate this code of conduct.
