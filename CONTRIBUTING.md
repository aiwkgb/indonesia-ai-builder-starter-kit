# Contributing Guide

Thank you for considering a contribution to Indonesia AI Builder Starter Kit.

This project exists to help Indonesian beginners build practical AI-assisted software projects using modern tools such as Claude, Codex, GitHub, and Vercel.

Catatan: project ini Indonesian-first. Kontribusi dalam Bahasa Indonesia sangat diterima, terutama jika membantu pemula memahami langkah berikutnya dengan lebih jelas.

## First-time contributor path

If this is your first contribution, start small:

1. Read [`README.md`](README.md)
2. Read [`docs/README.md`](docs/README.md)
3. Pick a small improvement
4. Open an issue or comment on an existing issue
5. Make a small focused change
6. Open a pull request and explain what you changed

## Good first contributions

Helpful first contributions include:

- Fixing unclear instructions
- Improving beginner explanations
- Adding practical examples
- Adding or improving checklists
- Improving bilingual summaries
- Suggesting new templates

## Contribution principles

Please keep contributions:

1. Beginner-friendly
2. Practical and execution-oriented
3. Relevant to Indonesian creators, students, solo founders, or small businesses
4. Clear enough for someone with limited coding experience
5. Honest and non-hype

## What to avoid

Please avoid:

- Big rewrites without discussion
- Unsupported claims
- Too much jargon
- Overengineering
- Promotional content

## Suggested template for new resources

When adding a new template or guide, include:

- Who this is for
- When to use it
- Step-by-step instructions
- Example output
- Common mistakes
- Next steps

## How to contribute

1. Fork this repository
2. Create a new branch
3. Make your changes
4. Open a pull request
5. Explain what you changed and why

## Code of conduct

Be respectful. This project is for learning, collaboration, and helping beginners grow.
