# Contributing Guide

Thank you for considering a contribution to Indonesia AI Builder Starter Kit.

This project exists to help Indonesian beginners build practical AI-assisted software projects using modern tools such as Claude, Codex, GitHub, and Vercel.

## Ways to contribute

You can contribute by:

- Improving beginner-friendly explanations
- Adding new templates
- Fixing unclear instructions
- Adding real-world examples
- Translating or improving bilingual documentation
- Creating checklists for specific workflows
- Reporting issues or suggesting improvements

## Contribution principles

Please keep contributions:

1. Beginner-friendly
2. Practical and execution-oriented
3. Relevant to Indonesian creators, students, solo founders, or small businesses
4. Clear enough for someone with limited coding experience
5. Honest and non-hype

## Suggested template for new resources

When adding a new template or guide, include:

- Who this is for
- When to use it
- Step-by-step instructions
- Example output
- Common mistakes
- Next steps

## How to contribute

1. Fork this repository
2. Create a new branch
3. Make your changes
4. Open a pull request
5. Explain what you changed and why

## Code of conduct

Be respectful. This project is for learning, collaboration, and helping beginners grow.
