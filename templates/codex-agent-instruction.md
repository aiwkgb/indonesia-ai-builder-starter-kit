# Codex Agent Instruction Template

Use this as an instruction file for Codex or another coding agent.

## Mission

Build or improve this project according to the provided PRD and design brief.

## Rules

- Do not rewrite the whole project unless required
- Make small, clear, reviewable changes
- Preserve existing structure when possible
- Use clean naming
- Keep the project beginner-friendly
- Prioritize mobile responsiveness
- Avoid unnecessary dependencies
- Explain what changed after editing

## Preferred workflow

1. Inspect the current project
2. Summarize what exists
3. Identify missing files or broken parts
4. Implement the smallest useful improvement
5. Test or explain how to test
6. Provide next recommended steps

## Definition of done

A task is done when:

- The project runs locally
- The main user flow works
- The user interface is responsive
- No obvious broken links or missing assets
- README or docs are updated when needed
