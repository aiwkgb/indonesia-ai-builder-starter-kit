# Landing Page Brief Template

Use this template before building a landing page.

This brief helps you explain the offer, audience, message, sections, and call to action clearly before writing copy or creating the page.

## How to use this template

1. Copy this file into your project.
2. Rename it to `docs/landing-page-brief.md`.
3. Fill in the sections below.
4. Keep the first version simple.
5. Review the brief before building the page.
