# Indonesia AI Builder Starter Kit

Open-source starter kit for Indonesian beginner AI builders.

> Bahasa Indonesia-first. Short English summaries are included to help international contributors understand the project context.

## Ringkasan Bahasa Indonesia

Indonesia AI Builder Starter Kit adalah starter kit open-source untuk pemula Indonesia yang ingin belajar membangun project dengan bantuan AI, GitHub, template, checklist, dan contoh project kecil.

Repo ini membantu creator, student, solo founder, small business owner, dan beginner builder untuk mulai dari brief, memilih template, mencoba contoh project, menyimpan progress di GitHub, lalu mengembangkan project secara bertahap.

## English summary

This repository is an early open-source educational starter kit for Indonesian beginner AI builders.

It provides practical templates, docs, checklists, and examples for AI-assisted building workflows using tools like Claude, Codex, GitHub, and Vercel.
