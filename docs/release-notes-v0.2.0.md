# Release Notes: v0.2.0

## Summary

v0.2.0 berfokus pada polishing template, memperjelas panduan untuk pemula, dan membuat repo ini lebih berguna sebagai starter kit untuk workflow project berbantuan AI.

Release ini bukan perubahan besar pada produk. Fokusnya adalah membuat dokumentasi, checklist, dan template lebih mudah dipakai oleh Indonesian beginner AI builders sebelum mereka mulai menulis brief, memakai Claude atau Codex, mengatur GitHub, atau deploy project kecil.

## What changed

Template dan checklist berikut dipoles agar lebih ringkas, jelas, dan beginner-friendly:

- PRD template
- Brand guideline template
- Claude project instruction template
- Codex agent instruction template
- Landing page brief template
- Vercel deploy checklist

Changelog juga diperbarui untuk mencatat progress v0.2.0 dan menjelaskan bahwa perubahan ini adalah bagian dari persiapan release berikutnya.

## Why this matters

Perubahan ini membantu Indonesian beginner AI builders dengan cara yang praktis:

- Brief project menjadi lebih jelas sebelum mulai membangun.
- Claude dan Codex lebih mudah diberi konteks, batasan, dan instruksi kerja.
- Workflow GitHub menjadi lebih rapi dan mudah direview.
- Dokumentasi project menjadi lebih praktis untuk pemula.
- First-time builders punya lebih sedikit kebingungan saat mulai dari ide, template, sampai deploy awal.

## Notes for contributors

Kontributor bisa membantu dengan:

- Memperbaiki dokumentasi yang masih kurang jelas.
- Menambahkan contoh yang praktis dan mudah dipahami.
- Memperbaiki penjelasan yang terlalu teknis.
- Menyarankan template baru yang beginner-friendly.
- Meninjau checklist agar tetap ringkas dan berguna.

Kontribusi kecil sangat membantu jika fokusnya jelas dan mudah direview.

## Honest project positioning

Project ini masih early-stage open-source educational project.

Repo ini belum mengklaim adopsi luas, jumlah pengguna besar, banyak download, atau impact besar. Fokus saat ini adalah membangun fondasi dokumentasi dan workflow yang jujur, praktis, dan berguna untuk pemula yang ingin belajar membuat project kecil dengan bantuan AI.
