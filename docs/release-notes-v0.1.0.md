# Release Notes: v0.1.0

## Initial open-source foundation

`v0.1.0` is the first foundation release of Indonesia AI Builder Starter Kit.

This release focuses on making the repository useful, understandable, and beginner-friendly for Indonesian creators, students, solo founders, small business owners, and beginner builders who want to build practical AI-assisted projects.

## Positioning

Indonesia AI Builder Starter Kit is an early open-source educational infrastructure project for Indonesian beginner AI builders.

It is designed to help beginners understand how to plan, document, build, save, and improve small AI-assisted projects using tools such as Claude, Codex, GitHub, and Vercel.

## Highlights

### 1. Beginner-first documentation

The initial documentation is written to help beginners understand the workflow before jumping into code.

The goal is to reduce confusion around GitHub, PRD, AI prompts, project structure, and iteration.

### 2. Indonesian AI builder glossary

The glossary explains common AI builder and open-source terms in simple Indonesian.

It helps beginners understand terms such as:

- PRD
- Brief
- Commit
- Issue
- Pull Request
- Deploy
- UI
- UX
- Workflow

### 3. Starter templates

The repository includes practical templates that beginners can copy and adapt for their own projects.

These templates help users create clearer project instructions before asking AI tools to build or edit files.

### 4. Beginner project structure example

The first example shows a simple folder structure for an AI-assisted project.

It explains where to put docs, assets, source code, and AI prompt instructions.

### 5. Open contribution foundation

The repo includes a contribution guide, code of conduct, roadmap, and issue templates so the project can grow in a transparent and beginner-friendly way.

## Who this release is for

This release is useful for:

- Indonesian creators learning to build with AI
- Students learning GitHub and project workflow
- Solo founders validating simple ideas
- Small business owners creating simple websites or landing pages
- Beginner contributors looking for documentation-first open-source practice

## What this release is not yet

This release is not yet a complete software framework, CLI tool, or production-ready app starter.

It is an early educational starter kit focused on documentation, templates, workflow, and beginner onboarding.

## Recommended next steps

After this release, the project can continue with:

- More real-world examples
- More bilingual documentation
- Improved template consistency
- More beginner-friendly issue labels
- Example landing page project structure
- Project audit checklist
