# v0.3.0 Documentation Roadmap

## Purpose

v0.3.0 focuses on improving the next layer of beginner-friendly documentation after the v0.2.0 documentation polishing release.

The goal is to make the project easier for Indonesian beginner AI builders to learn from, contribute to, and reuse in small practical projects. This includes clearer beginner examples, continued bilingual documentation, small prompt packs, contributor guidance, and simple learning paths.

## Current status

v0.2.0 has already been tagged and published as a GitHub Release.

The v0.3.0 cycle should build on that foundation carefully, without overstating adoption or expanding the project faster than the documentation can support.

## Priorities

- Improve beginner example project documentation.
- Continue bilingual documentation for important guides.
- Add small prompt packs for practical AI-assisted workflows.
- Improve contributor guidance for first-time contributors.
- Review issue templates for beginner clarity.
- Prepare simple learning paths for new AI builders.

## Non-goals

- No unsupported adoption claims.
- No large ecosystem impact claims.
- No claims that this project is already an established ecosystem dependency.
- No complex product features before the documentation foundations are stronger.
- No unnecessary file churn outside the planned documentation work.

## Suggested issue sequence

1. Improve beginner example project documentation.
2. Add bilingual summaries to key docs.
3. Add a simple prompt pack for AI-assisted project planning.
4. Review contributor guidance for first-time contributors.
5. Review issue templates for beginner clarity.

## Definition of done

- The v0.3.0 priorities are documented clearly.
- The roadmap stays honest, practical, modest, and beginner-friendly.
- The next issues are small enough for beginner contributors to understand.
- Documentation changes avoid unsupported adoption or impact claims.
- Planned work is limited to useful documentation, examples, prompts, and contributor guidance.
