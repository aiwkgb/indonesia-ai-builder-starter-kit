# Mulai dari Sini

Panduan ini dibuat untuk pemula Indonesia yang ingin mulai membangun project dengan bantuan AI.

## Tujuan

Setelah mengikuti panduan ini, kamu bisa:

- Membuat brief project yang jelas
- Menggunakan Claude atau Codex dengan instruksi yang rapi
- Menyusun struktur project di GitHub
- Menyiapkan publikasi project
- Mengembangkan project secara bertahap

## Alur sederhana

1. Pilih jenis project
2. Isi PRD
3. Isi brand guideline atau project brief
4. Masukkan instruksi ke Claude atau Codex
5. Buat atau edit project
6. Simpan project di GitHub
7. Publikasikan project
8. Minta feedback
9. Iterasi

## Pilih jenis project

Untuk produk digital:

- Gunakan examples/digital-product-landing-page
- Gunakan templates/prd-template.md

Untuk creator brand:

- Gunakan examples/creator-brand-website
- Gunakan templates/brand-guideline-template.md

Untuk local business:

- Gunakan examples/local-business-website

## Prinsip penting

Jangan mulai dari coding. Mulai dari brief.

AI akan lebih berguna kalau kamu memberi konteks yang jelas:

- Siapa target user-nya?
- Apa masalahnya?
- Apa hasil akhirnya?
- Seperti apa tampilannya?
- Apa saja batasannya?
