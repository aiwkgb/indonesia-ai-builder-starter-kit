# v0.2.0 Planning Note

This note captures a small, beginner-friendly direction for the next development cycle.

> Bahasa Indonesia-first. English summary included for international contributors.

## Ringkasan Bahasa Indonesia

Setelah `v0.1.0`, fokus `v0.2.0` adalah merapikan navigasi, dokumentasi bilingual ringan, dan pengalaman contributor pemula.

Prioritas kerja:

1. Pastikan README dan `docs/README.md` mudah dipakai sebagai pintu masuk.
2. Tambahkan contoh project yang lebih dekat dengan kebutuhan creator, student, solo founder, dan UMKM Indonesia.
3. Rapikan issue template agar contributor pemula tahu cara memberi masukan.
4. Tambahkan pola section reusable untuk landing page atau website sederhana.
5. Hindari klaim berlebihan tentang adoption, user, stars, atau impact.

## English summary

After `v0.1.0`, the `v0.2.0` focus is to improve navigation, lightweight bilingual documentation, and beginner contributor experience.

Work priorities:

1. Make README and `docs/README.md` easier to use as entry points.
2. Add examples that match Indonesian creators, students, solo founders, and small businesses.
3. Improve issue templates so beginner contributors know how to suggest changes.
4. Add reusable section patterns for landing pages or simple websites.
5. Avoid unsupported claims about adoption, users, stars, or impact.
