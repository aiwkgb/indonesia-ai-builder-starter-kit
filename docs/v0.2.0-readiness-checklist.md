# v0.2.0 Release Readiness Checklist

## Purpose

Checklist ini membantu maintainer meninjau repository sebelum menyiapkan release `v0.2.0`.

## Documentation readiness

- [ ] README links reviewed
- [ ] Changelog updated
- [ ] Release notes added
- [ ] Planning notes updated
- [ ] Maintenance log updated
- [ ] Claude Open Source application draft refreshed

## Template readiness

- [ ] PRD template reviewed
- [ ] Brand guideline template reviewed
- [ ] Claude project instruction template reviewed
- [ ] Codex agent instruction template reviewed
- [ ] Landing page brief template reviewed
- [ ] Vercel deploy checklist added

## Repository readiness

- [ ] Open issues reviewed
- [ ] Completed issues closed
- [ ] Latest commits pushed
- [ ] Working tree clean before release
- [ ] Release notes checked before tagging

## Positioning check

- [ ] No unsupported claims about users, stars, downloads, adoption, or ecosystem impact
- [ ] Project is described as early-stage
- [ ] Claims are based on existing files, issues, commits, changelog, and roadmap
- [ ] Language remains beginner-friendly and honest

## Next step

Setelah checklist ini selesai, maintainer bisa menyiapkan release tag `v0.2.0` atau melakukan final review pada release notes.
