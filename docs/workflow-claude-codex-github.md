# Workflow: Claude + Codex + GitHub

## Overview

This workflow helps beginners use AI tools without losing project direction.

## Step 1: Plan with Claude

Use Claude to create:

- PRD
- Brand guideline
- Project structure
- Writing draft
- Implementation plan

## Step 2: Build or edit with Codex

Use Codex for:

- Creating files
- Editing code
- Fixing bugs
- Refactoring
- Adding components
- Updating documentation

## Step 3: Save everything in GitHub

GitHub is the source of truth.

Keep:

- Code
- Documentation
- Templates
- Changelog
- Issues
- Roadmap

## Step 4: Publish your project

Use a simple publishing workflow so other people can review your work.

## Step 5: Iterate

After testing, create issues and improve the project step by step.
