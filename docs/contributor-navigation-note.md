# Contributor Navigation Note

This short note explains how contributors can navigate the repository.

> Bahasa Indonesia-first. English summary included for international contributors.

## Ringkasan Bahasa Indonesia

Gunakan repo ini dari folder utama:

1. Baca `README.md` untuk memahami tujuan project.
2. Buka `docs/README.md` untuk memilih dokumentasi.
3. Gunakan `templates/` untuk membuat brief project.
4. Gunakan `checklists/` untuk review.
5. Lihat `examples/` untuk contoh struktur project.

## English summary

Use this repository from the root:

1. Read `README.md` to understand the project.
2. Open `docs/README.md` to choose documentation.
3. Use `templates/` to create project briefs.
4. Use `checklists/` for review.
5. Explore `examples/` for project structure examples.
