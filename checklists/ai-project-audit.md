# AI Project Audit Checklist

Checklist ini membantu pemula mengecek project AI-assisted sebelum dipublish, dibagikan, atau dijadikan kontribusi open-source.

Gunakan checklist ini setelah project punya README, dokumentasi dasar, template, atau contoh project.

## 1. Project clarity

- [ ] Nama project jelas
- [ ] Tujuan project mudah dipahami
- [ ] Target pengguna disebutkan
- [ ] Masalah yang dibantu oleh project dijelaskan
- [ ] Scope project tidak terlalu melebar
