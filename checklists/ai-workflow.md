# AI Workflow Checklist

Before asking AI to build:

- [ ] Define project goal
- [ ] Define target user
- [ ] Define design direction
- [ ] Define required sections or features
- [ ] Define technical stack
- [ ] Define constraints
- [ ] Provide examples or references

When using AI:

- [ ] Ask for a plan first
- [ ] Give one task at a time
- [ ] Review output
- [ ] Test locally
- [ ] Commit changes
- [ ] Document important decisions
