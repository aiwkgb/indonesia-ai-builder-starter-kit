# AI Workflow Checklist

Checklist ini membantu pemula memakai AI sebagai asisten, bukan pengganti berpikir, mengecek, atau mengambil keputusan.

## 1. Before using AI

- [ ] Tujuan project sudah jelas
- [ ] Masalah dan target user sudah dipahami
- [ ] Output yang diinginkan ditulis dengan sederhana
- [ ] Batasan penting sudah dicatat

## 2. Prompt preparation

- [ ] Prompt memberi konteks yang cukup
- [ ] Prompt menjelaskan peran AI yang dibutuhkan
- [ ] Contoh atau referensi ditambahkan jika membantu
- [ ] Prompt meminta output step-by-step jika diperlukan

## 3. Reviewing AI output

- [ ] Jangan menyalin output AI tanpa dicek
- [ ] Cek apakah jawaban sesuai tujuan project
- [ ] Hapus bagian yang membingungkan atau tidak perlu
- [ ] Cek fakta, link, dan klaim sebelum dipublikasikan

## 4. AI-assisted coding or documentation

- [ ] Minta AI menjelaskan perubahan dengan bahasa sederhana
- [ ] Review kode atau dokumentasi sebelum commit
- [ ] Buat perubahan kecil dan fokus
- [ ] Jangan memasukkan data privat, API key, atau secret

## 5. Testing and feedback

- [ ] Test hasilnya secara manual
- [ ] Tanyakan apakah pemula bisa memahami output ini
- [ ] Catat batasan yang masih diketahui
- [ ] Minta feedback sebelum menyebut project selesai

## 6. Documentation habit

- [ ] Simpan prompt atau instruksi yang berguna jika relevan
- [ ] Dokumentasikan keputusan penting
- [ ] Sebutkan workflow berbantuan AI secara jujur jika berguna
- [ ] Jaga klaim project tetap realistis

## Final check

- [ ] Apakah AI membantu memperjelas pekerjaan?
- [ ] Apakah output final sudah direview manusia?
- [ ] Apakah project aman dibagikan sebagai versi awal?
